# AI Assistant

基于 Qt 6.9.0 和 DeepSeek API 构建的智能对话程序，支持多轮对话上下文管理、深度思考模式和个性化提示词定制。

## 🛠️ 技术栈

- **核心框架**: Qt 6.9.0
- **AI引擎**: DeepSeek
- **开发语言**: C++17
- **构建系统**: QMake
